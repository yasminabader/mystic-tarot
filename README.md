Overview: I made tarot reading system built on Notion MCP. My gamey/pixelated website lets users select a category and spread. 
The Notion MCP agent then draws cards from a 78-card Notion database, generates practical interpretations, logs readings to a journal, 
updates monthly analytics insights, and recommends a daily card with a Spotify song match. All powered by Notion as the brain.


I built an Automated Tarot Card Pull & Interpretation system using Notion MCP. The tech stack includes HTML/CSS for the frontend, Node.js 
for the backend, the Notion API to read/write to databases, and Cursor AI Agent with Notion MCP for intelligent automation.
I built this because as someone obsessed with tarot, I noticed two pain points: (1) I would forget long-term patterns in the cards I pulled, 
and (2) I struggled to understand what it meant when certain suits kept appearing. With Notion MCP, I can now track every reading automatically and surface meaningful insights over time.

Fullstack Website (3 screens):
1. Exterior of a pixel art Tarot Shop with an "Enter if you dare" button
2. Interior of the shop where the user selects a reading Category and Spread
3. Starry night screen where the Notion MCP Agent interpretation appears in real time

Auto-generated by Cursor AI Agent via Notion MCP:
1. Card of the Day - daily card pull with orientation, suit, arcana, a one-sentence daily forecast, a shadow work journal prompt, and a Spotify song recommendation matched to the card's energy
2. Weekly Tarot Insights - suit breakdown by percentage, most pulled card this month, top 3 cards overall, a 2-sentence pattern interpretation, and one concrete action item
3. Tarot Readings Journal - triggered by the website, each entry logs the reading date, category, spread (linked to Notion Spread Database), cards drawn (linked to Notion Tarot Card Database), card orientations, status, and a full Notion MCP interpretation with a next step

Note: The Notion Tarot Card Database contains all 78 cards with images, suit, and arcana type. The agent reads card meanings directly from Notion (not from general AI knowledge),  making every reading grounded in my personal database.
